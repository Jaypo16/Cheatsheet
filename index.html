HTML/CSS Cheat Sheet

Basic HTML Tags

Tag Name Description
<p> Paragraph The p tag defines a paragraph
< th > Table Header Defines a header in an HTML table
<footer> Footer Defines a footer for a document or section
<!DOCTYPE> Document Type Defines the document type
<br> Line break Inserts a single line break
<!--...--> Comment Defines a comment
<div> division Defines a division or a section in an HTML document

CSS Basic Properties

Property Description Values
color Sets the color of a text RGB, hex, keyword
line-height Sets the distance between lines Defines a header in an HTML table
letter-spacing Increase or Decrease the space between characters normal, length
text-align Aligns the text in an element left, right, center, justify
text-decoration Adds decoration to text none, underline, overline, line-through
text-indent Indents the first line of text in an element length, %
text-transform Controls the letters in an element none, capitalize, uppercase, lowercase
